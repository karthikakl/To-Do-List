# To-Do-List

Built a To-Do List application with interactive features for managing tasks. This project aimed at gaining practical understanding of React and enhancing skills in front-end development.

## Features
- **Add Tasks**: Users can add new tasks to the list effortlessly.
- **Remove Tasks**: Users can easily remove tasks that are no longer needed.
- **Rearrange Tasks**: Users can rearrange tasks in the list to prioritize their work.
- **Edit Tasks**: Users have the option to edit existing tasks for better management.
- **Flexible Task Prioritization**: Users can move tasks up or down in the list to adjust their priority.
